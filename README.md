## 🎮 ゲーム概要
風のスキルを使って相手を吹き飛ばし、ステージから落とすことを目的とした対戦アクションゲームです。  
風パワーゲージを管理しながら、状況に応じてスキルを使い分ける戦略性が特徴です。

## 🎮 操作方法

### 基本操作
| キー | 内容 |
|---|---|
| 矢印キー | 前後左右に移動 |
| スペースキー | シーン切替 |

### スキル操作  
※スキルは風パワーゲージを消費して使用します。

| キー | スキル名 | 内容 |
|---|---|---|
| Zキー | 突進スキル | 向いている方向に素早く移動（ゲージ5消費） |
| Xキー | 突風スキル | 近距離または遠距離の敵を吹き飛ばす（ゲージ10消費） |
| Cキー | 竜巻スキル | 周囲の敵全員を吹き飛ばす（ゲージ30消費） |

## 🚀 実行方法（Releases 版）
1. **Releases から ZIP ファイルをダウンロード**してください。  
2. **Okkochikara.exe と Resources フォルダーが同じ階層にある状態で起動**してください。  
3. フォルダー構造を変更しなければ正常に動作します。
